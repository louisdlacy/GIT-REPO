# ViewProps type

Represents the props of a [View()](https://developers.meta.com/horizon-worlds/reference/2.0.0/ui_view) component.

## Signature

```typescript
export declare type ViewProps = {
    children?: UIChildren;
    style?: ViewStyle;
};
```
